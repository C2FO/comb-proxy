# comb-proxy

## Overview

`comb-proxy` is plugin for comb to expose ProxyHelpers.

## Installation

```javascript
 npm install comb-proxy
```


## Usage

To initialize comb-proxy simply require it.

```javascript

var comb = require("comb-proxy");

```
